## Tool Specific

### NCodec

| Variable                                      | CLI Option | Default |
| --------------------------------------------- | ---------- | ------- |
| `NCODEC_TRACE_PATH`                           | _N/A_      | _None_ (trace disabled, path to write trace files) |
| `NCODEC_TRACE_PATH_{ecu_id}_{cc_id}_{swc_id}` | _N/A_      | _None_ (trace disabled, path to write trace files) |
